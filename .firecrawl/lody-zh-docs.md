# Lody [​](https://lody.ai/zh/docs\#lody)

Lody.ai 让你可以在手机或电脑上并行使用你的 Coding Agents 并与团队共享。

- 当前支持 Claude Code， Codex，OpenCode, Kimi 等所有支持 [ACP Registry](https://agentclientprotocol.com/get-started/registry) 的 CLI
- 通过 Worktree 来让 Coding 任务高效并行
- 更好的 GitHub 集成，自动绑定 PR，同步 review 评论
- 良好的移动端（PWA）的界面优化，手机也可以随时 vibe coding
- 对话完成时发送推送通知，不用担心忘记
- 团队共享 agent 设置和所有对话上下文，时刻保持信息一致

## 最新能力 [​](https://lody.ai/zh/docs\#%E6%9C%80%E6%96%B0%E8%83%BD%E5%8A%9B)

- [图像输入](https://lody.ai/zh/docs/image-input.html)：将图片作为会话上下文的一部分发送给 AI。
- [CLI 类型支持](https://lody.ai/zh/docs/cli-runtimes.html)：在一台机器上注册更多 CLI 运行类型。
- [上下文、Token 与额度](https://lody.ai/zh/docs/usage-and-quota.html)：在设置页查看上下文窗口余量、Token 花费和订阅额度。
